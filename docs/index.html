Project Summary

This project analyzes market sentiment from tweets and financial headlines related to Reliance Industries. Using NLP and sentiment scoring, we forecast stock prices via time series models. A live web app is available for real-time predictions.

The "Market Sentiment Analysis" project aims to predict stock price movements by integrating financial news sentiment with time series forecasting. Traditional technical analysis, though effective in capturing trends based on price and volume data, falls short in assessing the dynamic and often irrational impact of investor sentiment. Recognizing this gap, the project emphasizes the significance of market sentiment—shaped by continuous news updates—and its effect on Reliance Industries Limited (RIL), one of India’s largest conglomerates.

The project adopts a hybrid approach that blends explainable machine learning methods with robust statistical modeling. Initially, data was collected from the Economic Times, specifically filtering news articles relevant to RIL using Named Entity Recognition (NER) and keyword-based techniques. The textual data underwent rigorous preprocessing including tokenization, stopword removal, and lemmatization to ensure high-quality inputs for analysis.

For sentiment prediction, two approaches were implemented. In the first approach, a small subset of news articles was labeled using FinBERT, and topic modeling with LDA was used to extract topics. A traditional regression model like XGBoost was trained using these topic distributions to generalize sentiment prediction across the entire dataset. In the second approach, Word2Vec embeddings combined with TF-IDF weighted averaging were used to represent articles. The XGBoost model was trained on labeled financial news data to predict sentiment scores for RIL-specific articles. Feature engineering techniques such as PCA and Lasso Regression were explored but all features were retained for optimal performance.

The predicted sentiment scores were then aggregated on a daily basis and smoothed using an Exponentially Weighted Moving Average (EWMA) to create a sentiment time series. This time series was integrated as an exogenous variable into a SARIMAX model trained on RIL's historical stock price data from 2023 to 2025.

The SARIMAX model, enhanced with sentiment data, outperformed baseline models such as ARIMA and SARIMA. The model’s accuracy was validated using performance metrics like RMSE and MAPE. Visual comparisons also confirmed its ability to closely follow actual stock price trends.

For future forecasting beyond March 2025, a separate SARIMA model was trained to predict EWMA sentiment values. These predicted sentiment scores were then used in the SARIMAX model to forecast RIL stock prices, enabling a dynamic and sentiment-aware prediction pipeline.

In conclusion, the project effectively demonstrates the power of combining sentiment analysis with statistical forecasting. The successful application of this approach to RIL stock data highlights its potential for broader application across different sectors or companies. Future work could explore more advanced deep learning models such as LSTMs or Transformers to further enhance prediction accuracy.


Tech Stack

    - Python, NumPy, Pandas, Scikit-learn, NLTK
    - Sentiment Analysis using VADER
    -XGboost for sentiment-score
    - SARIMAX from statsmodels for time series forecasting
    - Streamlit for deployment

Methodology

- Scraped tweets and news headlines
- Cleaned and tokenized text data
- Assigned sentiment scores using VADER
- Applied EWMA to smooth scores
- Trained SARIMAX model with stock prices + sentiment

Model Evaluation: SARIMAX

Testing Accuracy: SARIMAX demonstrated better performance compared to ARIMA and SARIMA in terms of testing accuracy. Metrics such as Mean Squared Error (MSE), Root Mean Squared Error (RMSE), and Mean Absolute Percentage Error (MAPE) confirmed its predictive accuracy.

SARIMAX Model Evaluation:

  • MAE: 37.4163
  • RMSE: 43.9098
  • MAPE: 2.9738
